Install NeoVim first (see neovim-install.txt)

Prerequisites for LayzVim plugins I have:
sudo apt install tree-sitter-cli

Installation instructions:
https://www.lazyvim.org/installation

Rather than using the git repos in those instructions, if "config" is set up,
config checkout -- ~/.config/nvim

Instructions to set up "config" are in the "About" section of https://github.com/zymurgeek/dotfiles#
