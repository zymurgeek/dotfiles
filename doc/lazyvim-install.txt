= Cargo =
Note: /tmp is used by Cargo by default, but that directory is mounted noexec in runtimes.  Add this to ~/.cargo/config.toml:

[build]
target-dir = "cargo-target"         # path of where to place generated artifacts

cargo-target will be ~/cargo-target

= Testing =
Use script ~/bin/nvimt to test updates to NeoVim/LazyVim.  It uses the lazyvim-test directory instead of the nvim directory in these directories:
  ~/.cache
  ~/.config
  ~/.local/share
  ~/.local/state
The way it does this is by setting the environment variable NVIM_APPNAME to lazyvim-test.  nvim is the default.
