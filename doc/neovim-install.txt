Installation instructions:  https://neovim.io/doc/install/

TL;DR for Ubuntu:
cd
curl -LO https://github.com/neovim/neovim/releases/latest/download/nvim-linux-x86_64.appimage
chmod a+x nvim-linux-x86_64.appimage
sudo chown root:root nvim-linux-x86_64.appimage
sudo mv nvim-linux-x86_64.appimage /usr/local/bin
sudo ln /usr/local/bin/nvim-linux-x86_64.appimage /usr/local/bin/nvim
